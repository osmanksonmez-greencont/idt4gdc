# iDT4GDC — Intelligent Digital Twin for Green Data Centres

Official project website for **iDT4GDC**, a EUREKA ITEA4 research project (No. SUS2022-042) developing an intelligent digital twin platform to make data centres climate-neutral.

## Project Overview

iDT4GDC combines AI/ML optimisation, digital twin simulation, and real-time energy management to reduce the carbon footprint of data centre infrastructure. The platform integrates renewable energy forecasting, dynamic energy pricing, hardware-aware workload scheduling, and an interactive dashboard into a unified, production-ready system.

- **Programme:** EUREKA ITEA4
- **Project No.:** SUS2022-042
- **Duration:** 36 months (2023–2026)
- **ITEA4 Page:** https://itea4.org/project/idt4gdc.html
